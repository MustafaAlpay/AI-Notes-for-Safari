# AI Notes for Safari 📝🧠

Researching on the web can be overwhelming. To help with that, I developed **AI Notes for Safari**, a contextual intelligence sidebar.

It allows you to ask questions about the current webpage, extract key facts, and summarize long articles instantly. Everything is processed entirely on-device using **Apple Intelligence**, ensuring your browsing history and notes remain 100% private.

👉 **[Get it on the Mac App Store](https://apps.apple.com/us/app/ai-notes-for-safari/id6757966535)**

## Features
- **Contextual Q&A:** Ask questions directly related to the webpage you are viewing.
- **Instant Summaries:** Summarize long articles and extract key facts.
- **100% Private:** Processed entirely on-device via Apple Intelligence.
